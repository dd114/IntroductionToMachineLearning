Файлы digits.py и concrete.py являются запускаемыми, в первом происходит обучение и распознавание цифр датасета mnist, во втором же распознавание трещен в бетоне

Дополнительные пояснения:
- concrete.py
    -- border_selection() = функция для выделения границ обьектов (можно исползовать 2 ядра)

- mnist_loader.py = стандартные функции для чтения архивированного датасета mnist

- ./data = содержит картинки с изображением 2х видов цемента

- MLP.py = класс создания полносвязной нейронной сети, по названиям функций должно быть примерно понятно их предназначение. Код и математика бралась из многих сайтов, а также выводилась и писалась сама (основная статься для понимания работы https://habr.com/ru/companies/otus/articles/483466/)
